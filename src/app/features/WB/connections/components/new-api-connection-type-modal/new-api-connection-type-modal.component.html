После изменениня/добавления токена, подключение будет отключено. Вам необходимо будет првоести активацию с новыми данными.
Система проверит сервисы на чтение. Будьте внимательны, некоторые сервисы требуют запись данных. Но это невозможно проверить при активации. Поэтому во время работы, если сервер получит отказ несколько раз, ваше подключение будет отключено.
